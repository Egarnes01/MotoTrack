# Client Placeholder

The current MotoTrack MVP uses a lightweight browser frontend served from the Flask backend at:

- `server/app/static/index.html`
- `server/app/static/app.js`
- `server/app/static/styles.css`

This `client/` directory is included so your repo structure still reflects a standard full-stack project layout for capstone documentation.

If your team later upgrades to React, Vue, or another frontend framework, place it here.
